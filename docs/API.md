# 言页 API

稳定入口是`包:言页`，源码模块为`src/言序HTML.yx`。机器可读清单可用言序 1.1.12 的`文 --json`生成。

所有公开类都具有语言级`初始化`法。常规应用优先使用本页列出的工厂；直接构造类与对应工厂语义相同。

## 版本与默认限制

| 名称 | 类型 | 1.0 值 | 说明 |
| --- | --- | --- | --- |
| `版本` | `文` | `1.0.0` | 包的语义化版本。 |
| `默认最大渲染深度` | `数` | 64 | 根节点深度为 0。 |
| `默认最大渲染节点` | `数` | 10000 | 包含文档、组件、片段和其展开节点。 |
| `默认最大元素属性` | `数` | 256 | 每个元素的有效属性数。 |
| `默认最大输出字符` | `数` | 4194304 | 按 Unicode 字符计数。 |
| `默认最大输出片段` | `数` | 100000 | 非空写入回调次数。 |

`默认限制（）`返回上述五项的新字典。修改返回字典不会改变全局默认值。

`规范限制（限制：典）`把部分覆盖与默认值合并，并验证键、整数和硬上限：

| 键 | 最小值 | 硬上限 |
| --- | ---: | ---: |
| `最大深度` | 0 | 256 |
| `最大节点` | 1 | 100000 |
| `最大属性` | 0 | 1024 |
| `最大输出字符` | 1 | 16777216 |
| `最大片段` | 1 | 1000000 |

未知键、非数值、小数或越界值返回`HTML_LIMIT_CONFIG`。

## 错误协议

`错误详情（所误：误）：典`把库错误转换为：

```text
代码    稳定的 HTML_* 程序代码
消息    面向人的诊断，不作为程序协议
源代码  言序运行时原始错误代码
类别    言序运行时错误类别
位置    源位置
踪迹    调用踪迹
```

不是由言页产生的错误使用`HTML_RUNTIME`，原消息保留在`消息`中。

`尝试渲染（所节点：HTML节点）：典`不把渲染失败抛给调用方：

```text
成功时  {「成功」：真，「值」：文，「错误」：空}
失败时  {「成功」：假，「值」：空，「错误」：错误详情}
```

稳定错误代码：

| 代码 | 含义 |
| --- | --- |
| `HTML_RUNTIME` | 外部回调或运行时错误。 |
| `HTML_TAG` | 标签名语法不合法。 |
| `HTML_TAG_UNSAFE` | 标签被默认安全策略拒绝。 |
| `HTML_ATTRIBUTE` | 属性名、属性类型或属性位置不合法。 |
| `HTML_ATTRIBUTE_UNSAFE` | 属性被默认安全策略拒绝。 |
| `HTML_ATTRIBUTE_DUPLICATE` | 同一元素出现大小写不敏感的重复属性名。 |
| `HTML_URL` | URL 属性含空白/控制字符或策略外协议。 |
| `HTML_NODE` | 片段或元素收到非节点子项。 |
| `HTML_NODE_ABSTRACT` | 直接渲染基类或未支持的自定义节点。 |
| `HTML_VOID` | HTML5 空元素收到子节点。 |
| `HTML_COMPONENT` | 组件构建器没有返回`HTML节点`。 |
| `HTML_LIMIT_CONFIG` | 限制配置无效。 |
| `HTML_LIMIT_DEPTH` | 节点深度超过预算。 |
| `HTML_LIMIT_NODE` | 节点数量超过预算或构造硬上限。 |
| `HTML_LIMIT_ATTRIBUTE` | 元素属性数量超过预算或构造硬上限。 |
| `HTML_LIMIT_OUTPUT` | 输出字符数超过预算。 |
| `HTML_LIMIT_FRAGMENT` | 非空输出片段数超过预算。 |

## 节点类型

### `HTML节点`

所有节点的根类。`渲染（）：文`使用默认预算。直接创建并渲染基类会返回`HTML_NODE_ABSTRACT`。

### `HTML文字节点`

- 只读字段`内容：文`；
- `初始化（内容：文）`；
- `渲染（）：文`始终执行 HTML 转义。

使用`文字（内容：文）：HTML文字节点`创建。

### `HTML原始内容`

- 只读字段`内容：文`；
- `初始化（内容：文）`；
- `渲染（）：文`原样写出内容。

使用`原始（内容：文）：HTML原始内容`创建。这是显式可信入口，不解析或清洗输入；输出仍计入节点、字符和片段预算。

### `HTML扩展节点`

- `初始化（写出器：法）`；
- `写出（写入法：法）：空`调用受信任写出器；
- `渲染（）：文`通过统一渲染状态收集写出结果。

使用`扩展（写出器：法）：HTML扩展节点`创建。写出器接收一个`法（文）：空`式回调，每次调用都检查字符和片段预算。应用不应直接调用`写出`绕过正常渲染流程。

### `HTML片段`

`初始化（子节点：列）`验证每项是`HTML节点`并复制输入列。

| 成员 | 返回 | 说明 |
| --- | --- | --- |
| `添子（所节点：HTML节点）` | `HTML片段` | 追加节点并返回自身。 |
| `各子节点（）` | `列<HTML节点>` | 返回快照；修改结果不影响片段。 |
| `渲染（）` | `文` | 顺序渲染子节点，不增加包装标签。 |

使用`片段（子节点：列）：HTML片段`创建。片段自身计为一个渲染节点。

### `HTML属性`

- 只读字段`名称：文`；
- 只读字段`值：文|理|空`；
- `初始化（名称：文，值：文|理|空）`验证名称及 URL；
- `渲染（）：文`返回带前导空格的属性正文，或空文。

值语义：

| 值 | 输出 |
| --- | --- |
| `文` | ` 名称="已转义值"`；URL 属性再次校验。 |
| `真` | ` 名称`。 |
| `假` | 空文。 |
| `空` | 空文。 |

使用`属性（名称：文，值：文|理|空）：HTML属性`创建。

### `HTML元素`

- 只读字段`标签：文`；
- `初始化（标签：文，属性列：列，子节点：列）`验证并复制两份输入列。

| 成员 | 返回 | 说明 |
| --- | --- | --- |
| `添属性（所属性：HTML属性）` | `HTML元素` | 验证并追加；拒绝重复名。 |
| `添子（所节点：HTML节点）` | `HTML元素` | 追加节点；空元素拒绝。 |
| `各属性（）` | `列<HTML属性>` | 返回属性快照。 |
| `各子节点（）` | `列<HTML节点>` | 返回子节点快照。 |
| `渲染（）` | `文` | 使用默认预算渲染。 |

使用`元素（标签：文，属性列：列，子节点：列）：HTML元素`创建。

识别的 HTML5 空元素为`area`、`base`、`br`、`col`、`embed`、`hr`、`img`、`input`、`link`、`meta`、`param`、`source`、`track`和`wbr`。默认安全策略另行拒绝`base`和`embed`，因此它们不能通过普通`元素`工厂构造。

### `HTML组件`

- 只读字段`构建器：法`；
- `初始化（构建器：法）`；
- `构建（）：HTML节点`调用构建器并验证结果；
- `渲染（）：文`构建后通过统一渲染器输出。

使用`组件（构建器：法）：HTML组件`创建。组件节点和构建结果都会计入节点与深度预算；一次渲染只调用一次对应组件构建器。

### `HTML文档`

- 只读字段`根：HTML节点`；
- `初始化（根：HTML节点）`；
- `渲染（）：文`写出 doctype 和根节点。

使用`文档（根：HTML节点）：HTML文档`创建。固定前缀为`<!doctype html>\n`。

## 渲染法

| 法 | 返回 | 行为 |
| --- | --- | --- |
| `安全渲染（所节点：HTML节点）` | `文` | 使用全部默认限制。 |
| `渲染受限（所节点：HTML节点，限制：典）` | `文` | 合并限制并收集完整正文。 |
| `流式渲染（所节点：HTML节点，写入法：法，限制：典）` | `典` | 同步逐片回调，返回统计。 |

`流式渲染`结果：

```text
节点数  实际访问的节点数
字符数  成功写出的 Unicode 字符数
片段数  非空写入回调次数
```

回调异常会中止渲染并继续抛出。后续预算失败也会中止，但先前写出的片段不会回滚。需要原子正文时使用`渲染受限`，需要增量传输时由上层协议处理首部、错误和连接状态。

安全规则见 [SECURITY_MODEL.md](SECURITY_MODEL.md)，组合示例见 [GUIDE.md](GUIDE.md)。
