# yanxu-html API 参考

入口模块是`src/言序HTML.yx`，当前公开版本常量为`版本 = "0.1.0"`。

## 类型

### `HTML节点`

所有可渲染值的根类型。

| 成员 | 返回 | 说明 |
| --- | --- | --- |
| `渲染（）` | `文` | 输出当前节点的 HTML 表示。 |

### `HTML文字节点`

保存只读`内容：文`。`渲染`始终执行 HTML 转义。

### `HTML原始内容`

保存只读`内容：文`。`渲染`原样返回内容，不做验证或清洗；只用于已经审计的可信 HTML。

### `HTML属性`

保存只读`名称：文`与`值：文|理|空`。

| 值 | 渲染结果 |
| --- | --- |
| `文` | 输出` 名称="已转义值"`；URL 属性会额外校验。 |
| `真` | 输出` 名称`布尔属性。 |
| `假`或`空` | 不输出属性。 |

### `HTML元素`

保存只读`标签：文`以及可追加的`属性列：列`、`子节点：列`。

| 成员 | 返回 | 说明 |
| --- | --- | --- |
| `添属性（HTML属性）` | `HTML元素` | 追加属性并返回自身，便于链式组合。 |
| `添子（HTML节点）` | `HTML元素` | 追加子节点并返回自身；空元素会拒绝此操作。 |
| `渲染（）` | `文` | 按属性和子节点顺序输出元素。 |

空元素集合为`area`、`base`、`br`、`col`、`embed`、`hr`、`img`、`input`、`link`、`meta`、`param`、`source`、`track`、`wbr`。

### `HTML组件`

保存只读`构建器：法`。`构建（）`调用构建器并验证返回值是`HTML节点`；`渲染（）`构建后渲染。

### `HTML文档`

保存只读`根：HTML节点`。渲染时在根节点前添加`<!doctype html>\n`。

## 工厂

| 工厂 | 返回 | 用途 |
| --- | --- | --- |
| `文字（内容：文）` | `HTML文字节点` | 创建自动转义的普通文字。 |
| `原始（内容：文）` | `HTML原始内容` | 显式创建不转义的可信 HTML。 |
| `属性（名称，值）` | `HTML属性` | 创建文字、布尔或省略属性。 |
| `元素（标签，属性列，子节点）` | `HTML元素` | 创建并验证元素。 |
| `组件（构建器）` | `HTML组件` | 创建延迟构建组件。 |
| `文档（根）` | `HTML文档` | 创建 HTML5 文档。 |
| `安全渲染（HTML节点）` | `文` | 从节点根执行渲染。 |

## 失败代码

库以结构化错误消息前缀标记失败类别：

| 前缀 | 触发条件 |
| --- | --- |
| `HTML_TAG` | 标签名不符合白名单。 |
| `HTML_ATTRIBUTE` | 属性名不合法，或元素收到非属性值。 |
| `HTML_URL` | URL 含空白/控制字符或使用未允许协议。 |
| `HTML_NODE` | 元素子项不是 HTML 节点。 |
| `HTML_VOID` | 向 HTML5 空元素添加子节点。 |
| `HTML_COMPONENT` | 组件构建器没有返回 HTML 节点。 |

安全规则及其局限见[安全模型](security.md)。
