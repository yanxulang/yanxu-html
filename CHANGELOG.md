# 更新记录

本项目遵循语义化版本。1.x 保持公开名称、错误代码、默认安全策略和渲染语义兼容；安全边界的放宽需要新的主版本。

## 1.0.0 - 2026-07-18

### 破坏性变化

- 稳定中文包名改为`言页`；依赖键和导入从`yanxu-html`迁移到`言页`。
- 最低言序版本提高到 1.1.12。
- 默认拒绝脚本、样式、嵌入、SVG/Math 子语言标签及需要专用解析器的高风险属性。
- 大小写不敏感地拒绝重复属性。

### 新增

- 加入`HTML片段`、`片段`和快照隔离的组合 API。
- 加入`HTML扩展节点`与受预算约束的可信写出器。
- 加入`默认限制`、`规范限制`、`渲染受限`和真正逐片回调的`流式渲染`。
- 为深度、节点、属性、输出字符和输出片段提供默认预算与硬上限。
- 加入`错误详情`和不抛出的`尝试渲染`结构化协议。
- 扩展 URL 属性覆盖，并保持协议、空白和控制字符检查。
- 新增完整 HTML5 页面快照、边界规格、可执行示例与可校验性能负载。

### 工程

- 补齐兼容性、贡献、API、指南、架构、迁移、性能、安全模型和 1.0.0 发布文档。
- 使用机器 API 清单检查公开名称的文档覆盖。
- CI 固定言序 1.1.12，并验证格式、静态检查、规格、示例、基准、Release 构建、锁文件、权限和版本元数据。

## 0.1.1 - 2026-07-15

- 升级到言序清单格式 2，声明最低言序 1.1.5、默认公开导出、字节码构建目标和原生扩展禁用边界。
- 更新纯言序言包 0.2.0 的命令顺序与言序 1.1.5 CI 基线。

## 0.1.0 - 2026-07-14

- 建立 HTML 节点、文字、原始内容、属性、元素、组件和文档模型。
- 默认转义文字与属性值。
- 加入属性名、标签名、URL 协议和空元素安全约束。
